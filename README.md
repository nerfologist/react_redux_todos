# React/Redux todo app

Minimal app to showcase interaction between React and Redux.

Showcases the following concepts:

- redux:
  - store
  - actions
    - passing actions to components as props
  - reducer
    - combining reducers
- react:
  - functional components
- react-redux
  - provider
  - container

# How to use

```sh
npm install
npm serve

(head over to localhost:8080)
```
